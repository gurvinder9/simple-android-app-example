## Getting Started

Each project is self-contained with its own build configuration and dependencies. Navigate to the specific project folder to work with it.

### Opening in Android Studio

1. Open Android Studio
2. Select "Open an existing project"
3. Navigate to the specific project folder (e.g., `project1/`)
4. Wait for Gradle sync to complete
5. Run on an emulator or physical device

## Requirements

- Android Studio (latest version recommended)
- JDK 8 or higher
- Android SDK with API level 24+
